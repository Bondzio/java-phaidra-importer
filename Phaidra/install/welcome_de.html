Il Phaidra Importer è un’applicazione Open Source integrata con la piattaforma Phaidra che consente di caricare libri, collezioni e video.

Nota bene: I file compatibili per tipo di oggetto digitale sono:

  • libro: TIF, JPG, PNG XML (generati dal Phaidra Importer)
  • collezione: TIF, JPG, PDF, PNG, XML (generati dal Phaidra Importer)
  • video: AVI, MPEG, MP4

Per installare il Phaidra Importer, segui i passi da 1 a 6.
Nel caso di problemi, contatta l’help desk di Phaidra.
